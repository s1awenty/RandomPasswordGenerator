# Random Password Generator

Автор: ппп ппп

Описание:
Это графическое приложение для генерации случайных паролей с настройками длины и типа символов. Пароли отображаются на экране и сохраняются в историю, которая загружается при запуске. Приложение использует библиотеку `random` для генерации паролей и сохраняет историю в JSON-файл.

Особенности:
- Настройка длины пароля с помощью ползунка
- Выбор типов символов через чекбоксы
- Генерация пароля по выбранным параметрам
- История сгенерированных паролей отображается в таблице
- История сохраняется между сессиями

Примеры использования:
1. Установите Python 3.x
2. Запустите файл `password_generator.py`
3. Настройте параметры и нажмите "Генерировать"
